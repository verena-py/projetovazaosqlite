Projeto Vazao

Tem como objetivo, agir como uma ferramenta para análises e uma melhor caracterização da Bacia Hidrográgica do Rio Coruripe, Alagoas.

# Configurações do projeto
1 Linguagem Python
2 Framework web utilizado: Django
3 Bibliotecas principais: Pandas, HidroComp, HidroPythonAPI, Plotly, Bootstrap
4 Desenvolvido pelo software Pycharm
5 Modelo de banco de dados: Observations Data Models 2 (ODM2)
6 Banco de dados baseado SQL, implementado pela bilioteca SQLite
7 Documentações de referência: django.org; odm2.org.
